Widget de reservas

Para adicionar no site:

O widget 'div' pode ser utilizado em qualquer domínio, desde que contenha o ID da suíte que possui a reserva.

Caso o motel disponha apenas de reservas imediatas, o widget será exibido no mobile.

  • Primeiro, é necessário adicionar esses links CSS no <head></head> 
                
                    <link rel="stylesheet" href="https://go.guiademoteis.com.br/go-sites/go.min.css">
                    <link rel="stylesheet" href="widget.css">

**Baixe o arquivo CSS, onde é possível alterar as cores do widget: baixar

  • Depois, inclua este código onde deseja que o widget seja exibido. 
                    
                        <div class="suites-availability-go" data-suite="ID DA SUÍTE"></div>
  • Coloque este link JS antes do fechamento do </body> 
                    
                        <link rel="stylesheet" href="https://go.guiademoteis.com.br/go-sites/go.min.js">
